loveUAD
AI-powered dementia caregiver support app
Healthcare

The Problem

Dementia caregivers have no structured digital support system. Medication management is disorganised, patient data privacy is routinely compromised, and caregivers have no access to evidence-backed AI guidance at 2am when they need it most.

What I Built

Medication reminders with prescription OCR scanning. AI dementia guidance backed by a large research paper database. Anonymous 17-digit patient codes — no name, no email, no data breach risk. Built on Google Cloud Run with PostgreSQL.

PythonGoogle Cloud RunPostgreSQLOCRGemini APIFlutter

Achieved

  • Accepted into NHS Propel HealthTech Accelerator 2025
  • Zero-knowledge patient data model — anonymous 17-digit codes
  • Privacy-first architecture designed for NHS Digital integration
  • Prescription OCR scanning working in production
02
WritenDraw
Chat-based workplace simulation assessment platform
Education / HR

The Problem

Traditional online assessments can be gamed. Candidates copy-paste, cheat, or perform in ways that bear no resemblance to real workplace behaviour. Employers have no way to test how someone actually thinks on the job.

What I Built

A chat-based simulation that puts candidates inside workplace scenarios. No voice — pure conversation. The AI tracks reasoning, response patterns, and retake behaviour. Deployed on Google Cloud Run using the Gemini API.

Gemini APIGoogle Cloud RunPythonChat Interface

Achieved

  • Pilot: 27 sessions, 10 real candidates
  • 44% retake rate — engagement data in the wild
  • 16.2% average score — baseline established
  • 0 paste attempts detected across all sessions
  • Submitted to Gemini Live Agent Challenge
03
CareCircle
Zero-knowledge health notification system
Healthcare

The Problem

Patient health notifications require personal data — mobile numbers, emails, real names. Every system that collects this data is a data breach waiting to happen. NHS Digital needs a model where patient privacy is structural, not policy.

What I Built

An AI-powered health notification system using zero-knowledge architecture. Patients register anonymously — no mobile number, no email, no real name. The data breach risk is eliminated by design, not by compliance documents.

Zero-Knowledge ArchitectureAIPythonNHS Integration Patterns

Achieved

  • Academic paper published on Zenodo
  • Ongoing feasibility discussion with a major UK healthcare provider
  • Live at carecircle.top

TrueSkills
Anti-cheating AI assessment for education
Education

The Problem

AI has made cheating trivially easy. Students submit AI-generated work as their own. Standard proctoring tools play cat-and-mouse with students instead of measuring genuine understanding.

What I Built

An assessment system that tests personalised understanding — questions tailored to each student so generic AI answers don't work. Built with advanced proctoring and detection. Also includes an AI tutoring layer for UK Year 2–6 children.

SilentPulse
Safety app disguised as a clock
Social Impact

The Problem

Vulnerable women in abusive situations can't safely call for help — abusers monitor their phones, check their apps, watch for anything that looks like a safety tool. A recognisable safety app is dangerous to have.

What I Built

An app that looks like a clock. Opens like a clock. Works like a clock. But underneath, it sends silent distress signals. Built in Flutter, distributed via Firebase App Distribution in partnership with West Yorkshire NGOs.

FlutterFirebaseFirebase App Distribution

Achieved

  • Deployed in partnership with West Yorkshire NGOs
  • Disguised UI — zero visual footprint as a safety app
  • Silent distress signal working in production

NarrativeLink
Semantic search over historical slave narratives
History / AI

The Problem

What I Built

A semantic search engine that understands meaning, not just keywords. Ask a question in plain English, get back the relevant narratives. Built and deployed on Google Cloud Run for the Google Cloud Run Hackathon.

Semantic SearchGoogle Cloud RunPythonVector Embeddings
